Q1: Why might we consider using Git?

Answer: Git enables version control, which allows us to manage and track changes in code. It also is a great tool for collaborating with peers, 
as updated code can easily be accessed. In addition, in relation to this class, it is easy to submit work through git.

Q2: What field could data science be applied to? Give an Example.

Answer: One field data science could be applied to is healthcare. For example, data from previous outbreaks can be entered, which could then analyze patterns and potentional outcomes
if there were to be another outbreak in the future. This would help for better preperation and understanding of what to expect.
